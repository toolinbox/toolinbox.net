在 iPic 中添加腾讯云 COS 图床

腾讯云对象存储（COS: Cloud Object Storage）是面向企业和个人开发者提供的高可用，高稳定，强安全的云端存储服务。更多介绍，可参考 官方介绍

在 iPic 中添加腾讯云 COS 图床

打开 偏好设置，进入 图床 页，选择添加 腾讯云 COS

以下分别介绍各项的意义及如何配置：

  • Bucket
    • 即腾讯云 COS 中对应的 Bucket
  • Access Key 与 Secret Key
    • 即腾讯云 API 密钥管理中的 SecretId 与 SecretKey
  • 网址前缀
    • 用于生成上传图片的完成 Http 路径，比如 https://ipic-1251234567.cossh.myqcloud.com
    • 如果你绑定了域名，可以使用绑定的域名

完成输入后，可以点击 验证 按钮。如果输入没有问题，则右侧会出现 通过 链接，点击后就可以看到测试时上传的图片。

除了上述基本的设置，腾讯云 COS 还支持几项高级设置。点击 网址前缀 右侧的高级设置图标，即可弹出如下设置界面：

  • 网址前缀
    • 和前文介绍的一致
  • 文件名前缀
    • 可简单理解为腾讯云 COS 中的目录
    • 例如，如果希望 iPic 上传的图片都位于 blog 目录中，只要在这里输入 blog，所有上传的图片则均位于 blog 目录中，生成的图片链接类似于：https://ipic-1251234567.cossh.myqcloud.com/blog/pic.jpg
  • 自定义文件名
    • 支持以下三种文件名
    • 纯文件名 也即和上传时文件名相同，比如 pic.jpg
    • 日期-文件名 也即和之前的 iPic 相同，比如 2016-06-16-pic.jpg
    • 随机 比如 jk8l1.jpg，可以大大缩短链接长度
  • 网址后缀
    • 即在网址最后追加自定义的内容，为以后扩展预留

最后，点击 应用 按钮进行保存。

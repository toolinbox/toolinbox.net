iPic 更新日志

发表于 2016-05-22 | 分类于 iPic

v1.5.3

2018/03/29

  • 优化上传图片至默认图床

v1.5.0

2017/10/23

  • 支持上传图片至腾讯云 COS
  • 可直接拖拽 GIF 图片上传

v1.4.8

2017/05/19

  • 问题修复与性能改进

v1.4.0

2016/12/21

  • 上传前压缩图片
  • 全面适配 https
  • 支持 iPic Mover，一键迁移 Markdown 中所有图片

v1.3.1

2016/10/18

  • 七牛支持华南、北美节点
  • 可以复制图床
  • 内嵌 iPicUploader 上传服务
  • 适配 Swift 3.0

v1.2.0

2016/08/29

  • 新增 Amazon S3 图床
  • 修复 Bug
    • 批量上传图片时，链接顺序与选择的一致
    • 从 Dock 栏文件夹中拖动上传有时没反应

v1.1.2

2016/08/21

  • 新增 阿里云 OSS 图床
  • 优化七牛上传速度

v1.1.1

2016/08/01

  • 批量上传时保存批量结果
  • 新增快捷键上传已复制图片
  • 切换链接类型移至主菜单
  • 七牛、又拍图床支持自定义文件名
  • Imgur、Flickr 支持多个账户
  • 精简主菜单
  • 支持 macOS Sierra

更多介绍

v1.1.0

2016/07/03

  • 正式上架 Mac App Store

v1.0.7

2016/07/01

  • 支持上传至自己的 Imgur 账户

v1.0.6

2016/06/29

  • 上传至微博时增加图片类型后缀

v1.0.5

2016/06/27

  • 支持沙盒模式
  • 调整菜单栏图标

v0.8.0

2016/06/22

  • 支持上传图片至 Flickr
  • 可以在 Finder 中右击图片、使用 iPic 上传
  • 支持拖动图片至菜单栏中 iPic 图标后上传
  • 七牛云支持图片样式
  • 又拍云支持自定义版本

v0.7.6

2016/06/11

  • 支持上传图片至 又拍云

v0.7.1

2016/05/31

  • 支持上传图片至 Imgur
  • 七牛云支持文件名前缀（可理解为“文件夹”）
  • 记录更多上传历史，退出时保存
  • 支持开机自启动

v0.6.0

2016/05/25

  • 支持上传至 七牛
  • 菜单栏图标：自动上传时闪烁、上传时显示进度
  • 支持使用快捷键打开菜单

iPaste - 简洁高效的剪切板神器

发表于 2011-11-07   |  

简洁、高效

iPaste 只出现在菜单栏中，交互高效不花哨；甚至可以隐藏菜单栏图标，全程快捷键操作。

iPaste 默认快捷键：

  • 使用 ⇧⌘V 显示剪切板历史
  • 使用 ⌃⌘V 粘贴为纯文本

当然，可以在偏好设置中修改为自己喜欢的快捷键。

强大

可钉住常用的剪切板历史；进而固定其快捷键，实现批量粘贴时的盲操作。
支持文本、图片、文件等任意格式，适应所有 macOS Apps，不服来战。

聪明

支持开机自启动。
自动排除 KeyChain、1Password 等安全软件。

下载

相信你会非常享受 iPaste 的简洁、高效，马上拥有她吧！

Klib - Kindle、Apple Books、多看标注管理

发表于 2017-11-28 | 分类于 Klib

Klib 是全新的读书笔记管理工具，目前已支持 Kindle、Apple Books、多看。

为笔记而生：

  • 隐藏已读：专注于当下阅读
  • 回顾笔记：在 Kindle for macOS 中查看上下文，全球首创
  • 方便分享：以「Markdown」格式复制、导出至「印象笔记」(Evernote)
  • 自由编辑、全面搜索：书名、作者、笔记、备注等

地表最强导入：

  1. 通过 USB 从「Kindle」导入
  2. 通过网络从「亚马逊」导入
  3. 导入「Kindle for iPhone」等客户端的标注笔记
  4. 导入 Apple Books 中的 Highlights、Notes
  5. 导入「多看」系统中的书摘
  6. 使用 macOS 服务快速收集书摘

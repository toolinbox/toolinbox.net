使用 Dropbox 同步 Klib 数据库

在 Klib 实现通过 iCloud 在不同 Mac 电脑同步前，可以借助 Dropbox 等第三方工具来同步。这里以 Dropbox 为例进行介绍，假定 Dropbox 本地同步目录位于：~/Sync/Dropbox/

  • 打开「终端」Terminal
  • 在 Dropbox 中创建「Klib」文件夹
    • mkdir -p ~/Sync/Dropbox/Klib/
  • 将 Klib 数据库目录链接到 Dropbox 中
    • ln -s ~/Library/Containers/net.toolinbox.iKindle/Data/Library/Application\ Support/Klib/data ~/Sync/Dropbox/Klib/data
  • 在另一台 Mac 电脑做相同的设置，就可以在多台 Mac 电脑间同步 Klib 数据库了。

提示：Klib 的数据库为 SQLite 格式，位于下面的目录中，可在 Finder 中按下 Command + Shift + G、输入路径后打开。

~/Library/Containers/net.toolinbox.iKindle/Data/Library/Application\ Support/Klib/data
